# ModernCodingTheory
# 现代编码理论/ModernCodingTheory

* 该项目是2025年春季同济大学研究生课程《现代编码理论》的代码部分
* 项目代码结构
  * ModernCodingTheory/
    ├── coding_final/                  # 项目核心代码所在目录
    
    │   ├── compra_codings/           # 比较不同编码方案的模块的代码实现
    
    │   ├── conv_code/                # 卷积码实现
    
    │   ├── cyclic_code/              # 循环码实验实现
    
    │   ├── hamming/                  # 汉明码相关实现
    
    │   ├── linear_block/            # 线性分组码实现
    
    │   ├── figures/                 # 绘图、结果图像
    
    │   ├── vectorizer_figures/      # 矢量化图片
    
    ├── README.md                    # 项目概览与使用指南
    
    └── LICENSE                      # 项目的开源协议声明
